Este projeto é um site de aluguel de imóveis, pesquisa e aluguel de imóveis. Os usuários podem cadastrar imóveis para aluguel (desde que tenham uma conta), enquanto os usuários podem visualizar os imóveis disponíveis.

Funcionalidades feitas
Tela de Início: Página inicial que apresenta o site.
Tela de Cadastro: Cadastro de usuário. (não está totalmente completa)
